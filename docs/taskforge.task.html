taskforge.task package¶

Provides the Task and Note classes used throughout Taskforge.

Module contents¶

class taskforge.task.Task(title, id=None, context='default', priority=1.0, notes=None, created_date=None, completed_date=None, body='')[source]¶

Represents a task in a Task List.

This class is the basic unit in Taskforge and is central to all functionality.

The basic instantiation of a Task only requires a title and will fill out any required metadata with default values:

>>> from taskforge.task import Task
>>> Task('An example Task')
Task(c659687d9ad54b308a258850a5a06af1)

All fields available for a task and their defaults are:

Parameters:
  • title (str) – The title or ‘summary’ of a task.
  • id (Optional[str]) – The unique id for this task. If None this will be generated using uuid.uuid4(). Should only be provided if deserializing an existing Task.
  • created_date (Optional[datetime.datetime]) – A datetime object representing when this task was created. If not provided defaults to datetime.now(). Should only be provided if deserializing an existing Task.
  • body (str) – Default (“”) - The body or ‘description’ of a task.
  • context (str) – Default (“default”) - The ‘list’ this task belongs to. Common values are work, personal etc.
  • priority (float) – Default (1.0) - The priority of this task, this is the primary sorting criteria for tasks.
  • notes (List[Note]) – Default (None) - A list of Note objects to for this task.
  • completed_date (datetime.datetime) – Default (None) - A datetime object representing when this task was completed.

class taskforge.task.Note(body, id=None, created_date=None)[source]¶

A note or ‘comment’ on a task.

A basic note instantiation only requires the body field. All other fields are optional and id should not be set unless instantiating from an existing Note.

Parameters:
  • body (str) – The body of the note.
  • id (Optional[str]) – The unique id for this note. If None this will be generated using uuid.uuid4(). Should only be provided if deserializing an existing Note.
  • created_date (Optional[datetime.datetime]) – The created_date for this note. If None this will be generated using datetime.datetime.now(). Should only be provided if deserializing an existing Note.
